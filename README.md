1. Nombre del proyecto:
proyecto  3D de aula 2do Año  

2. Descripción general:
 Es un proyecto desarrollado  que permite visualizar y manipular objetos 3D de forma interactiva usando tecnología web.

3. Objetivo:
Aprender el manejo de entornos 3D y mostrar objetos tridimensionales en una aplicación accesible desde el navegador.

4. Características:
 Permite rotar, acercar, alejar y mover los modelos 3D.
Interfaz sencilla con controles intuitivos.
 Compatible con distintos formatos de modelos (.obj y glb).

5. Tecnologías:
HTML, CSS, JavaScript, Three.js.

6. Instalación y ejecución:
Descargar o clonar el repositorio y abrir el archivo index.html en un navegador moderno.

7. Uso:
El usuario puede interactuar con los objetos 3D usando el ratón o gestos táctiles.
